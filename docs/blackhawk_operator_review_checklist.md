# Blackhawk Operator Review Checklist

Use this checklist before Phase 8 regression.

- Can Will identify who is waiting on him from each actionable card?
- Can Will open the original source in one click when a source link exists?
- Can Will create a task from an email or meeting candidate without hunting?
- Are meetings understandable without expanding the card?
- Are missing sender, source, or calendar metadata gaps shown truthfully?
- Is any card relying on filler, generic summary, or fake certainty?
